Rules and Formats

Please read the rules carefully!

  • The platform, Hackerrank, supports a variety of programming languages (C/C++, Java, Python are preferred). See programming languages supported here
  • This is an individual contest. No collaboration or any forms of help from another person!
  • No help from A.I. or ChatGPT are allowed. We expect you to follow the academic integrity and honesty.
  • However, if you encounter anything that you are unfamiliar with, you can always search it up on the internet. The process of thinking is the most important step to ace the contest!
  • There will be a number of problems in increaing difficulty for you to solve. You can try them in any order.
  • You can submit solutions unlimited times. Points will NOT be taken off by submitting incorrect codes.
  • If you have any questions about the problem statements during the competition, feel free to message one of the admins in discord (Join!) We will be there throughout the event!
  • IMPORTANT: The starting template for any challenges in NPC does not include any additional part besides the very basic main function. The participants are expected to write their own input processings. It is strongly recommended for you to search how to process input in your language of choice!!

Scorings

Please read the scoring and grading policy carefully!

  • Each problem contains a certain number of test cases to test the output of your code. Each problem is worth certain points, and these points are going to be distributed evenly among all test cases. So passing more test cases = more points on that problem!
  • The problems are arranged in ascending order of difficulty. The more difficult, the more points a problem is worth!
  • The sum of the scores on each problems will be one's final score.
  • Again, you can submit solutions unlimited times without having points taken off for wrong submissions!
  • Participants are ranked by score. If two or more participants achieve the same score, then the tie is broken by the total time taken to reach the best score.
